# Лендинг Центра R&D Сбера | Нижний Новгород

**Современная визитная карточка центра исследований и разработки, созданная на React + Vite.  
Проект объединяет интересы новых сотрудников, резидентов коворкинга и партнёров в едином цифровом пространстве и полностью соответствует брендбуку Сбера.**

---

## Ключевой функционал

*   **Интерактивный чек-лист новичка** — помогает пройти первую неделю без стресса, с прогресс-баром и анимацией конфетти при 100% выполнении.
*   **Умный FAQ-виджет** — быстрый доступ к ответам на 9 частых вопросов: спортзал, бронирование, адаптация, программа лояльности и др.
*   **Живая карусель с отзывами** — центральное мнение и боковые карточки с инициалами, строгий стиль с жёлтыми звёздами и общим рейтингом 5.0.
*   **Мероприятия с раскрытием** — главные анонсы центра с тегами, описанием и кнопкой «Участвовать».
*   **Карточки возможностей центра** — спортзал, коворкинг, обучение, лояльность, мерч — с анимированными цифрами и ссылками на сервисы Сбера.
*   **Адаптивный дизайн** — корректное отображение на всех устройствах, мобильное меню-портал, плавные анимации Framer Motion.
*   **Доступ без авторизации** — лендинг полностью публичный, не требует входа во внутреннюю сеть.

---

## Брендинг и технологии

*   **Шрифты:** SB Sans Display (Semibold, Light) — используются для заголовков. Основной текст — Roboto.
*   **Цвета:** Фирменный зелёный Сбера (`#34C924`), жёлтый для звёзд (`#FFB800`), стекломорфизм (glassmorphism) в Header при скролле.
*   **Логотип:** SVG-логотип Сбера с автоматической инверсией цвета на тёмном фоне Hero.
*   **Анимации:** Framer Motion для появления секций, плавных переходов, карусели отзывов и прогресс-бара.
*   **Конфетти:** Canvas Confetti — срабатывают при 100% выполнении чек-листа.

---

## Технический стек

*   **React 19** (Vite + React-плагин)
*   **Framer Motion** (анимации секций и компонентов)
*   **Canvas Confetti** (эффект завершения чек-листа)
*   **React Intersection Observer** (для отслеживания видимости блоков)
*   **CSS Modules** (изоляция стилей)
*   **React DOM Portals** (для мобильного меню вне иерархии)
*   **GitHub Pages** (для демонстрации, опционально)

npm install
